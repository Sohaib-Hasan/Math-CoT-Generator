## 🛠 Methodology: The "Gold Standard" Rubric
Each reasoning trace in this repository is evaluated against a strict 4-point rubric:
1. **Accuracy:** 100% verified via symbolic computation.
2. **Atomic Logic:** Every logical bridge is explicitly documented; no hidden steps.
3. **Format Integrity:** Strict adherence to `<think>` tags and benchmarking markers (`#### Result`).
4. **Metacognition:** Documentation of assumptions, traps, and "Aha Moments" that guide a model toward the correct answer.
