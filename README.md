# AI-Math-Reasoning-Portfolio: Gold Standard Dataset Engineering

This repository showcases **professional-grade Chain-of-Thought (CoT) dataset engineering** for training Large Language Models (LLMs) in **mathematical reasoning**.  
All samples follow rigorous standards for **Supervised Fine-Tuning (SFT)** workflows and demonstrate **accuracy, logical rigor, and reproducibility**.

---

## Overview

As an **AI Math Training Specialist**, I develop "Gold Standard" reasoning traces to **eliminate model hallucinations** and strengthen logical reasoning.  
This portfolio demonstrates:

- **Metacognitive Reasoning:** Planning, self-correction, and structured <think> blocks.  
- **Mathematical Rigor:** Step-by-step derivations with LaTeX formatting.  
- **Symbolic Verification:** Automated Python (SymPy) scripts ensuring **100% accuracy**.

## Dataset Samples

The repository contains **10 diverse reasoning samples**, including:

- **Arithmetic & Logic (GSM8K):** Multi-step word problems (rates, proportions, financial math).  
- **Advanced Algebra & Calculus (AIMO):** Olympiad-level problems (arithmetic sequences, multivariable optimization, exponent comparisons).  
- **Symbolic Verifiers:** Python scripts to validate each reasoning trace.  

> **Note:** A Colab notebook is included for easy **interactive exploration and testing** of each dataset sample.

---

## Gold Standard Rubric

Each reasoning trace is evaluated against a strict **4-point rubric**:

1. **Accuracy:** Verified via symbolic computation (SymPy).  
2. **Atomic Logic:** Every logical step explicitly documented.  
3. **Format Integrity:** Adheres to <think> tags and benchmark markers.  
4. **Metacognition:** Notes assumptions, traps, and insights guiding correct solutions.
